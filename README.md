# Library Management System (Python)

## Overview
This is a simple Library Management System built using Python. The project demonstrates file handling, data storage using JSON, and basic program logic for managing books and users.

## Features
- Add and manage books
- Store data using JSON files
- Simple menu-driven system
- Beginner-friendly structure for learning Python

## Technologies Used
- Python
- JSON (for data storage)
